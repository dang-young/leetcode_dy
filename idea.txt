# data structure

* set vs map
- set: set of data
- map: set of key, value

* map/set vs unordered map/set
- unordered: 원소정렬X/hash 이용 -> fast
- 단, 원소의 개수가 많아 해시충돌이 많이 발생하는 경우 오히려 속도저하 가능성이 있음
